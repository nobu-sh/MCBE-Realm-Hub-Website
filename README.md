## Getting Started

#### Install Deps
```
$ npm install
```

####  Run Dev Server
```
$ npm run dev
```

#### Build
```
$ npm run build
```
#### Start Production
```
$ npm start
```

When running dev server client is accessible at `http://localhost:6969` and server is accessible at `http://localhost:6970`

When running in production mode everything is served at `http://localhost:6970`
